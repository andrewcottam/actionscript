this file is a descendent class from the site content box wclass hich introduces custom behavior for all site tool content boxes and descendents the class has a tool header and a loading box the tool header shows the tool name, e.g. site species list the loading box is used to show when the tool is loading and is dynamically bound to the application variable toolinprogress therefore, whenever a tool is run, the loading box is shown and when the tool stops the loading box is made invisible the loading property is used to set the visibility of any UI components in any instances of contentboxsitetool for example, while the tool is loading, no tables or controls are shown when their visibility property is bound to the loading property but when loading has finished, the results of the query can be shown in the controls which will not be visible
